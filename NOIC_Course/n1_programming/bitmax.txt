# Manipulação de bits

Checar se o bit j de um numero n está ligado.
basta olhar se n & 2^j é maior que 0
Na prática:

bool ligado(int n, int j)
{
	if(n & (1 << j)) return true;
	else return false;
}

Ligar o bit j de um numero;

void ligar(int &n, int j)
{
	n = n | (1 << j);
}

Você tem x questões. Você estimou a dificuldade da i-ésima questão como ci. Agora você tem que preparar uma prova com essas questões tal que a dificuldade total das questões seja pelo menos l e no máximo r e que a diferença entre a questão mais difícil e a mais fácil seja pelo menos x. Calcule a quantidade de maneiras de preparar a prova.

#include <bits/stdc++.h>
using namespace std;
const int inf = 1e9 + 10;


int n, l , r, x;
int v[20];
long long resp;

int main()
{
	ios::sync_with_stdio(false), cin.tie();

	cin >> n >> l >> r >> x;

	for(int i = 0; i < n; i++)
		cin >> v[i];
    // preenchendo o vetor v - prova com 

	for(int mask = 0; mask <= (1 << n) – 1; mask++)// percorro todas as bitmasks
    //1 << n é uma operação de deslocamento de bits (bitwise shift). Ela pega o número 1 (em binário, 0001) e "empurra" os seus bits n posições para a esquerda, preenchendo os espaços vazios à direita com zeros.
    // Na prática, essa expressão é uma forma extremamente rápida de calcular a potência de 2 elevada a n (ou seja, \(2^{n}\))
	{
		long long s = 0; // oq tem esse s?

		int maior = 0;

		int menor = inf;

		for(int i = 0; i < n; i++)// percorro todos os bits
		{
			if(mask & (1 << i)) //checo se a questão está na prova que estou olhando
            // Quando y é uma potência de 2, a operação X & y (operador bitwise AND) serve para verificar se o bit na posição correspondente a y está ativado (vale 1) no número X.
			{ 
				s += v[i];
				maior = max(maior, v[i]); // calculo a maior dificuldade - função intessante, mais rapida que if x < y max = y

				menor = min(menor, v[i]); // calculado a menor dificuldade
			}
		}

		if(s >= l and s <= r and (maior – menor) >= x) resp++; // se a prova é valida, adiciono um no contador
		// ou seja, testar TODAS as provas possíveis para ver se é valida porém fazemos o teste por bitmask, acelerando
	}
	cout << resp << "\n";
	return 0;
}
